# Emulator User Guide

The Emulator allows you to run your generated configurations in a software simulation environment. This is useful for debugging and functional verification before running on actual hardware.

## Features
*   Cycle-accurate simulation (or functional simulation, verify based on actual implementation).
*   Trace generation.
*   State inspection.
