# Project T-Stack Documentation

Welcome to the documentation for the T-Stack project. This project consists of three main components:

1.  **Program (Compiler)**: Tools for building computational graphs and generating hardware configurations.
2.  **Emulator**: A simulation environment for validating configurations and functional correctness.
3.  **Analyser**: A performance analysis tool for evaluating the execution on hardware.
